Leetcode_有多少...

Leetcode

给你一个数组 nums，对于其中每个元素 nums[i]，请你统计数组中比它小的所有数字的数目。

换而言之，对于每个 nums[i] 你必须计算出有效的 j 的数量，其中 j 满足 j != i 且 nums[j] < nums[i] 。

以数组形式返回答案。

示例：

思路：

先将数组排序，那么比当前数字小的数量就是排序后数组里该数字的索引，于是想到用map来保存每一个数字的索引，其中关键字为数字，值为数字在排序数组中的索引。考虑到排序数组里面有相同的元素，所以只存放相同数字里的第一个数字的索引。
