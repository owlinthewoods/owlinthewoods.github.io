Leetcode_最小高度树

Leetcode

给定一个有序整数数组，元素各不相同且按升序排列，编写一个算法，创建一棵高度最小的二叉搜索树。

示例：

给定有序数组: [-10,-3,0,5,9],

一个可能的答案是：[0,-3,9,-10,null,5]，它可以表示下面这个高度平衡二叉搜索树：

     0 
    / \ 
  -3   9 
  /   / 
-10  5

思路：

既然数组是有序的，那么就从数组的左，右半部分别组成根节点的左子树和右子树，于是便想到递归，递归时传入原来数组的一半。
