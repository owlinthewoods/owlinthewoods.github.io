Leetcode_化栈为队

Leetcode

实现一个MyQueue类，该类用两个栈来实现一个队列。

示例：

思路：

建立两个stack，其中stack1保存入栈的元素，元素push到stack1里面。当需要peek操作时，依次将stack1中的元素pop，然后加入到stack2当中去。如果stack2为空，那么就先需要进行转移的操作，然后peek。如果stack2不为空，直接对stack2进行peek操作即可。进行pop操作时也是对stack2进行pop，同样stack2为空时也需要先将stack2的数据转移到stack1当中去再pop。
