Thinking in Java

3.4 赋值

对于基本数据类型，赋值就是复制内容。对于引用数据类型，复制的是引用即地址。

3.7 关系操作符

如果对对象使用==运算符，那么比较的是引用。如果要比较两个对象的实际内容是否相同，可以使用所有对象都适用的特殊方法equals，但是如果对于自定义的类对象使用该方法，比较的仍然是引用。该方法可以重写。
